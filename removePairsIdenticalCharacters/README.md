#  Удалить пары одинаковых символов, которые стоят рядом

_Задание на собеседовании в компанию [Альфа-Банк](https://alfabank.ru/)_

### Задача

Дана строка.

Необходимо удалить пары одинаковых символов, которые стоят рядом.

### Пример

| На входе   | На выходе |
|------------|-----------|
| cccabbaddd | cd        |

# Найти первое не повторяющееся число в массиве

_Задание на собеседовании в компанию [IT_ONE](https://www.it-one.ru/) 17.12.2022_

### Задача

Дан массив целых чисел.
Нужно найти первое не повторяющееся число в массиве.

### Пример

| На входе | На выходе |
|-----------------|-----------|
| 9, 4, 9, 6, 7, 4 | 6         |

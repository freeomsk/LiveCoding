# Реализовать метод, который вернет коллекцию без дубликатов

_Задание на собеседовании в компанию [Сбер](http://www.sberbank.ru)_

### Задача

Дан массив целых чисел.

Реализовать метод, который вернет коллекцию без дубликатов.

### Пример

| На входе      | На выходе |
|---------------|-----------|
| 1, 2, 1, 2, 3 | 1, 2, 3   |
